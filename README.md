# PAPS
Репозиторий с лаб. работами по проектированию архитектуры программных систем

Выполняет Безводинских Эдуард Владимирович

**Тема проекта**: Разработка информационной системы для поддержки процесса приоритизации процессов импортозамещения на основе данных статистики внешнеэкономической деятельности РФ.

**Суть проекта**: Разработка приложения для поддержки принятия решений по финансированию отечественного производства конкретных категорий товаров из классификатора ТН ВЭД (Товарной номенклатуры внешнеэкономической деятельности). Производится ранжирование товарных категорий на основании данных внешней статистики с использованием методов MCDA (multi criteria decision analysis). Сначала по формулам рассчитываются критерии, затем с помощью AHP определяются веса критериев для WSM и затем WSM вычисляет итоговый результат, используемый для ранжирования товарных категорий (6-значных кодов ТНВЭД).
